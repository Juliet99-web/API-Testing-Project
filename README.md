# API-Testing-Project- Booking API

## Project Descripion
This project demostrates API testing using postman on a Booking API
The aim of this project is to validate the functionality , reliability , and perfomance of REST API endpoints.

## Tools Used
-postman
-REST API
-Javascript( postman tests)
-Github

## Test validation performed
- Status code validation
- response time validation
- Response body validation
- data type validation
- schema validation

  ## How to run the project
  1. Download the collection JSON file from this reporatory
  2. import it into postman
  3. Run the requests using the collection runner.
